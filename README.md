🐾 PHP Project: Курсовая работа

Курсовой проект находится в стадии активной разработки.

🚀 Текущий прогресс
На данный момент реализован следующий функционал:

✅ Каталог: Полный список объектов с удобным просмотром.

✅ Адаптивная верстка: Корректное отображение на мобильных устройствах и планшетах.

✅ Динамические страницы: Реализованы переходы на подробные карточки животных с использованием GET-запросов.

✅ Интеграция с БД: Главная страница полностью формируется на основе данных из MySQL.

⏳ Информационные страницы: Частично готовы. В процессе настройки вывода специфических данных из базы.

🛠 Перечень технологий

Язык программирования: PHP 8.x

БД: MySQL

Веб-сервер: Apache

Фронтенд: HTML5, CSS3 ( + адаптация)
